Install the module in your Nuxt app with nuxi:

<!--more-->

```bash
npx nuxi module add nuxt-precognition
```

## Manual installation

1.  Install the with your package manager

```bash
pnpm add add nuxt-precognition
```

2.  Add module to your project `nuxt.config.ts`

```typescript
export default defineNuxtConfig({
  modules: ['nuxt-precognition'],
  precognition: {

  }
})
```
